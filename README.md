# 📦 Retail Sales Forecasting & Inventory Optimization System

## 🚀 Overview

This project predicts future retail sales and optimizes inventory levels using machine learning and business logic.

## 🎯 Problem Statement

Retail businesses often face:

* Overstocking → increased cost
* Stockouts → lost sales

## 💡 Solution

This system:

* Forecasts demand
* Calculates safety stock
* Generates reorder recommendations

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* Streamlit

## 📊 Features

* Sales forecasting
* Inventory optimization
* Reorder point calculation
* Visualization dashboard
